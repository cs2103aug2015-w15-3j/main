## Changelog 

### Basic Features
------------------

1. ``Add`` 
 - with tag(s)
 - with priority 
 - multiple tasks with similar deadline 
 - task to a parent task
 
2. ``Edit`` 
 - task's time only 
 - priority
 
3. `Display`
 - tasks on specific date
 - overdued tasks
 - floating tasks
 
4. ``Done``
 - with tag
 - multiple ids support
 
5. ``Delete``
 - with tag
 - multiple ids support
 
6. ``Search`` 
 - with tag 
 
7. ``Set``

### GUI 
--------
1. Formatted display view 
 - color coded based on priority
 - length limit 
 
2. Activate & Hide toggle with keyboard shortcut, Ctrl+H 

3. Simple & Advanced view when minimize or maximize 

### Extra Features
------------------
1. Autocomplete 
 - task name 
 - command
 - tag
 
2. Shortcuts 
 - undo, `Ctrl+Z`
 - redo, `Ctrl+R`
 - clear input, `Ctrl+C`

3. Return previous command in inputbar with `Up`
